# Palavras-Chave Prioritarias do Nicho
> O Guardiao Sobrio — Luis Vanzer
> Atualizado em: Junho 2026

Esta lista foi construida a partir de tres fontes:
1. Vocabulario real do publico brasileiro que fala sobre alcool e sobriedade nas redes
2. Termos de busca com volume e intencao no contexto de recuperacao
3. Alinhamento com os pilares da marca (ESPELHO . TATICA . ESCUDO)

---

## Nivel 1 — Palavras de Alta Intencao (Dor Explicita)

Sao os termos que a pessoa digita quando JA SABE que tem um problema ou esta em crise.
Usar em: titulos de videos, primeiras 3 palavras do hook, bio.

| Palavra / Frase | Por que funciona |
|---|---|
| parar de beber | busca mais direta do nicho no Brasil |
| como parar de beber | intencao de acao imediata |
| sobriedade | identidade — quem ja se reconhece no processo |
| primeiros dias sem beber | dor aguda, busca nos primeiros 30 dias |
| 30 dias sem alcool | desafio mensuravel, muito buscado |
| alcoolismo | termo tecnico mais buscado para o tema |
| dependencia do alcool | variante tecnica com alta busca |
| gatilhos para beber | dor especifica e taticamente forte |
| recaida no alcool | momento de maxima vulnerabilidade + busca ativa |
| como nao beber numa festa | situacao concreta e muito buscada |

---

## Nivel 2 — Palavras de Identidade (Pertencimento)

Sao termos que a pessoa usa quando ESTA SE IDENTIFICANDO com o processo.
Usar em: hashtags, subtitulos, frases de abertura do video.

| Palavra / Frase | Por que funciona |
|---|---|
| sobrio | identidade simples e direta |
| sobriedade ativa | diferencia de "apenas nao beber" |
| vida sem alcool | estilo de vida, nao doenca |
| recuperacao | processo, nao cura |
| 1 dia de cada vez | cuidado: associado a AA — usar com cautela ou ressignificar |
| dia 1 | potente nos primeiros dias, muito buscado no TikTok |
| jornada de sobriedade | narrativa de processo, nao destino |
| alcoolista em recuperacao | auto-identificacao honesta |
| ex-bebedor | identidade nova — evitar tom de derrota |

---

## Nivel 3 — Palavras de Publico Secundario (Familiares)

Sao termos buscados por quem esta DO LADO de alguem que bebe.
Usar em: videos e protocolos especificos para conjuge/familia.

| Palavra / Frase | Por que funciona |
|---|---|
| familiar de alcoolista | busca direta do publico secundario |
| como ajudar quem bebe | intencao de acao do familiar |
| conjuge de alcoolista | dor muito especifica e pouco servida |
| codependencia | termo tecnico com alta busca entre familias |
| como falar com quem bebe | situacao concreta de conflito |
| Al-Anon | referencia de grupo de apoio para familias |
| Amor-Exigente | referencia de grupo focado em limites |
| limites com quem bebe | dor pratica do familiar |

---

## Nivel 4 — Palavras do Metodo (Posicionamento da Marca)

Sao termos que DIFERENCIAM O Guardiao Sobrio de outros criadores do nicho.
Usar em: bio, descricao de produtos, apresentacao da marca.

| Palavra / Frase | Por que funciona |
|---|---|
| guardiao sobrio | marca propria — construir associacao |
| metodo guardiao | nome do metodo proprio |
| espelho tatica escudo | pilares memoraveis e proprios |
| sobriedade com estrategia | diferenciacao: nao e so espiritualidade ou AA |
| sobriedade sem religiao | diferenciacao importante para publico secular |
| estoicismo e sobriedade | nicho cruzado com alto engajamento |
| mindfulness e alcool | nicho cruzado com busca crescente |
| saude mental e alcool | contexto amplo, alto volume |
| masculinidade e sobriedade | nicho especifico e pouco explorado no Brasil |

## Hashtags Recomendadas por Tipo de Video

> Regra de ouro para TikTok: 3 a 5 hashtags por video — misturar 1 de alcance geral, 2 de nicho especifico, 1 de identidade da marca.

### Videos de Choque de Realidade (ESPELHO)
```
#sobriedade #comoparardebeber #alcoolismo #guardiaosobrio #vicioemalcool
```

### Videos de Tatica / Protocolo (TATICA)
```
#primeiros30dias #sobriedadeativa #comoparardebeber #guardiaosobrio #recaida
```

### Videos de Acolhimento / Familia (ESCUDO)
```
#familiadealcoolista #sobriedade #comoajudarquembebe #guardiaosobrio #codependencia
```

### Videos de Historia Pessoal / Vulnerabilidade
```
#jornadadosobriedade #sobriedade #dia1 #guardiaosobrio #vidasemalcool
```

## Palavras a EVITAR (Risco de Posicionamento Errado)

| Palavra | Motivo |
|---|---|
| 12 passos / AA | Associacao religiosa — a marca e secular |
| cura do vicio | Promessa nao sustentavel — vicio nao tem cura, tem manejo |
| Deus / fe / oracao | Vinculo espiritual que nao e o posicionamento da marca |
| ex-vicioso | Tom autodepreciativo, nao alinha com identidade sobria |
| motivacao / motivacional | Tom de coach — exatamente o que a marca NAO e |
| heroi / vencedor / guerreiro | Cuidado: romantiza demais — a marca fala da trincheira, nao do pedestal |
